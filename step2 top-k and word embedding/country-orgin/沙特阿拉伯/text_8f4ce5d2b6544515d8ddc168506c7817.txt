现代重工控股公司 ( HHIH ) 是现代重工集团 ( HHI Group ) 的控股公司，该公司与沙特阿拉伯国家石油公司 ( Saudi Aramco ) 签署了一份关于氢和氨相关业务合作的谅解备忘录。通过该协议，双方正在寻找氢、氨研发的机会。现代 Oilbank HHIH 的炼油厂单位 , 计划与沙特阿拉伯国家石油公司合作实现一个 " 零碳 " 的过程 , 和韩国造船和海洋工程 ( KSOE ) 造船 HHIH 单位 , 计划开发了世界上第一个结合船能够携带液化石油气 ( LPG ) 货物和捕获的二氧化碳 ( CO2 ) 。根据一份谅解备忘录，现代石油银行将从沙特阿美进口液化石油气，然后将其转化为蓝色氢气。这些氢气将用于现代石油银行 ( Hyundai Oilbank ) 炼油厂的脱硫设施，并将用于市场上的汽车和发电厂燃料。此外，现代石油银行 ( Hyundai Oilbank ) 计划到 2040 年在韩国各地开设 300 个氢充电站，从而建立一条销售氢的供应链。它还将推广使用氨燃料的业务。现代石油银行计划从沙特阿拉伯国家石油公司接收蓝氨，并将其用作液化天然气 ( LNG ) 锅炉的燃料，该锅炉计划在 2024 年建成。该公司表示，如果将氨用作电厂燃料，二氧化碳排放量将显著减少，使整个过程更加环保。双方还将加强在造船业务方面的合作。KSOE 将成为第一家开发出既能装载液化石油气货物又能同时捕获二氧化碳的船舶的造船商。公司还致力于氨气运输船和推进船的开发。沙特阿美技术服务高级副总裁 Ahmad a . Al Sa 'adi 表示 :" 通过这份谅解备忘录，沙特阿美和 HHI 集团建立了战略伙伴关系，在蓝氢和蓝氨领域发现新的雄心勃勃的商业机会，这是未来的能源燃料。"" 这项合作有望通过碳捕获过程帮助减少温室气体排放。"来源：GAS COMPRESSION magazine ，氢云链编辑整理
